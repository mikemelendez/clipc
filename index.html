ClipC

Unicode-focused test utilities in one place.

Tools

      Welcome to ClipC

      A utility website for QA and test engineers to quickly generate, transform, encode, and validate unusual Unicode and edge-case input.

      The layout uses a left-hand menu inspired by the browsing style of Material Design's content navigation.

      I want to...

      Convert text to Unicode styles

      Quickly produce strange copy/paste test strings.

      Generate long test input

      Create length markers, repeated strings, and Unicode ranges.

      Encode or decode payloads

      Use Base64, URL, hex, binary, and hash tools.

      Create random test data

      Generate random email, phone, card-like, and address values.

      Copy WebSec payloads

      Quickly grab common XSS and SQL injection test strings.

      Sections

      Unicode Text Converter

      Transliterate plain text into obscure Unicode variants.

      Unicode Text Transformations

      Braille conversion, IDN encode/decode, interleaving, and zalgo text.

      Generate Text

      Generate deterministic identifiers, full character ranges, and repeated strings.

      Encode / Decode

      Base64 and URL encoding utilities.

      Common Input Generator

      Randomized email, phone, card-like and address-like test values.
